# Instalación del S.O Ubuntu 22 LTS

El primer paso que tendremos que realizar será la descarga de la ISO de Ubuntu 22, junto con una herramienta para poder crear unidades USB flash.

Enlace de Ubuntu 22 LTS [ISO](https://ubuntu.com/download/desktop)
Enlace USB flash [Rufus](https://rufus.ie/en/)

Una vez lista la unidad USB comenzaremos la instalación, en el primer paso tendremos que seleccionar el idioma del S.O.

![IMG](https://user-images.githubusercontent.com/67869168/195138859-431642ae-a4e7-469f-afc7-b86c9f0dcc4f.JPG)

## Actualizaciones y otro software

En este caso para que la instalación sea lo más ligera y rápida posible elegiremos la opción de instalación mínima, después instalaremos nosotros el software que necesitemos.

Para las actualizaciones las descargaremos al instalar el S.O, ya que podría provocarnos algún error si no las instalamos

![IMG](https://user-images.githubusercontent.com/67869168/195139249-62832667-1173-4bf8-915a-2df77156d085.JPG)

## Tipo de instalación

Podríamos realizar una instalación avanzada del sistema, pero como en este caso es solo un S.O para escritorio, no realizaremos ninguna partición, aremos una instalación simple.

![IMG](https://user-images.githubusercontent.com/67869168/195139721-bbbed1e1-771b-4d8e-9149-879181137f25.JPG)
